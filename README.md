## 🇬🇧 English
**Ethiopia Web Application** is a dynamic web project developed as a term paper for the **Internet Programming II** course at **Sakarya University of Applied Sciences - Sakarya Vocational School**. The application introduces Ethiopia's cultural, geographical, and historical features with a content management system.

### Tech Stack
* **Framework:** .NET Framework 4.7.2
* **Architecture:** ASP.NET MVC
* **Database:** MSSQL (Entity Framework 6)
* **Language:** C#
* **Frontend:** HTML5, CSS3, Bootstrap, JavaScript

### Installation & Setup
1. **Clone** the repository to your local machine.
2. Create the database using the `database_script.sql` file provided in the root directory.
3. Open the **`Web.config`** file and update the **Connection String** according to your local SQL Server instance.
4. Build and run the project using Visual Studio.

## 🇹🇷 Türkçe
**Etiyopya Tanıtım Projesi**, **Sakarya Uygulamalı Bilimler Üniversitesi - Sakarya Meslek Yüksekokulu** bünyesindeki **İnternet Programcılığı II** dersi için dönem ödevi olarak geliştirilmiştir. Proje, Etiyopya'nın kültürel, coğrafi ve tarihi zenginliklerini tanıtan dinamik bir web uygulamasıdır.

### Kullanılan Teknolojiler
* **Platform:** .NET Framework 4.7.2
* **Mimari:** ASP.NET MVC
* **Veritabanı:** MSSQL (Entity Framework 6)
* **Dil:** C#
* **Önyüz:** HTML5, CSS3, Bootstrap, JavaScript, jQuery

###  Kurulum ve Çalıştırma
1. Projeyi bilgisayarınıza indirin.
2. Ana dizindeki `database_script.sql` dosyasını kullanarak veritabanını oluşturun.
3. **`Web.config`** dosyasındaki **Connection String** bilgisini kendi yerel SQL Server ayarlarınıza göre güncelleyin.
4. Projeyi Visual Studio üzerinden derleyip çalıştırın.
